PowerPrompt

IMPORTANT: This is not made by me. I've only contributed to it many times, so that's why I put it here. PowerPrompt is made and should be associated with Mihael2017. Besides, he allowed to make my website an offical PowerPrompt source, so here we are.

An awesome new way to get things done! Introducing: PowerPrompt! A console-based assistant with bunch of important commands, shortcuts to most important programs, such as system settings, Windows accessories and counting! It also comes with many functional things, like a to-do list, safe password keeper, disk partitioning tool, file organization commands, including a character counting tool for text files. If you don't really like messing with command line arguments, PowerPrompt also has a slightly more user-friendly GUI!
