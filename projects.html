PowerPrompt

Created by Mihael2017
An awesome new way to make things done! Introducing: PowerPrompt! Your new source of computer management. Version that is published here won't be the newest. Newest version is shared with a limited group, but we still want to share something with you, so here we are. PowerPrompt includes life-changing features like:

  • Window management
  • Easy access to all of Windows Administrative Tools, as well as Windows Accessories, Settings app and Control Panel
  • Password Keeper
  • Disc partitioning
  • File organization
  • Command history
  • Character counter
  • To-do list
  • Voice controlling (coming soon)
  • Exporting and importing settings
